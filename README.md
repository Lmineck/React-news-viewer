# 📰 React-news-viewer 📰  
책 '리액트를 다루는 기술'의 '외부 API를 연동하여 뉴스 뷰어 만들기'를 참고한 클론코딩 프로젝트입니다  
외부 API 'https://newsapi.org/' API 키를 발급받아 사용합니다  
뉴스를 카테고리 별로 분류하여 서비스하는 기능을 제공합니다
